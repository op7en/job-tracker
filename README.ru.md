# Job Tracker — Трекер Откликов

Fullstack веб-приложение для отслеживания откликов на вакансии в процессе поиска работы.

## Функциональность
- Регистрация и авторизация пользователей
- Добавление, обновление и удаление откликов
- Отслеживание статуса (Отклик, Собеседование, Отказ, Оффер)
- Данные сохраняются в базе данных

## Стек технологий
**Фронтенд:** React, TypeScript, Tailwind CSS, Axios, React Router  
**Бэкенд:** Node.js, Express, TypeScript, PostgreSQL, JWT, bcrypt  
**Деплой:** Vercel (фронтенд) + Railway (бэкенд + база данных)
